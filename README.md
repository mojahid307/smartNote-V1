
  # SmartNote

  ## Features

  - **Text notes:** Create and edit plain or formatted notes with a clean editor.
  - **Hand-drawn notes:** Draw sketches and annotate with the drawing canvas and expanded drawing view.
  - **Photo notes:** Add and organize images using the photo grid and expanded photo viewer.
  - **Voice notes:** Record, play back, and manage voice memos.
  - **Rich editing & expand views:** Edit notes in-place or open expanded views for focused editing.
  - **Search:** Fast searching across notes to quickly find content.
  - **Authentication:** Login and signup flows are included for multi-user scenarios.
  - **Sync & backend:** Integrated with Supabase and server functions (see `supabase/functions/server`) for storage and synchronization.
  - **Responsive UI:** Modern component library and responsive layouts for desktop and mobile.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
